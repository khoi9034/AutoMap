Local GIS request engine for draft map review

Turn plain-English county GIS requests into reviewed recipes, packets, WebMap JSON, and local previews without publishing anything.

1. Create Recipe Parse geography, topics, filters, layer roles, and confidence.
2. Generate Review Packet Write local recipe, WebMap JSON, warning, and review files.
3. Preview Map Inspect the draft in a local browser map before any publish step.
4. Apply Adjustments Edit YAML adjustments and preserve the original packet.
5. Approve Packet Reviewer sign-off creates a separate approved packet when warnings are addressed.
6. Dry-Run Publish Create a receipt only. No ArcGIS item is created from the UI.
