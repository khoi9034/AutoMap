Controlled Real Publish

Real private draft publishing is CLI-only in v1.3 and disabled by default. It requires an approved packet, --confirm-publish, AUTOMAP_ALLOW_REAL_PUBLISH=true, and AUTOMAP_PUBLISH_DRY_RUN=false.

The UI does not offer one-click real publish. AutoMap never publishes publicly, never shares to the organization, and never overwrites or deletes ArcGIS items.

Portal Smoke Test

The UI can run a dry-run smoke test only. Real smoke testing is CLI-only and can create at most one private draft Web Map item when all safety flags are enabled.

CLI real smoke test:

python -m app.main --portal-smoke-test {{ approved_path }} --confirm-publish

Required local flags: AUTOMAP_ALLOW_REAL_PUBLISH=true and AUTOMAP_PUBLISH_DRY_RUN=false. Cleanup is manual in Portal.
